## Current Release

### Version 1.1.0 (Released)
Highlights delivered in 1.1.0:
- Full Edit Mode dragging support with proper lock state handling
- Inline Edit Mode settings dialog (compact + advanced tabbed interface)
- Visual anchor positioning for cooldown and keybind text
- Collision warning for overlapping text anchors
- Improved settings layout and taint-safe edit mode behavior

## Future Releases

### Version 1.5.0 (Planned)

#### Native WoW Controller Support
- 📋 **Roadmap**: Native controller keybind detection
  - Detect WoW's native controller bindings
  - Display controller button icons for bound spells
  - Support gamepad mode detection
  - Work without requiring any controller addons

#### Gamepad Icon Support
- 📋 **Roadmap**: Controller icon rendering
  - Display Xbox/PlayStation controller button icons
  - Auto-detect controller type when possible
  - Fallback to generic gamepad icons
  - Support for Xbox, PlayStation, and generic controllers

#### ConsolePort Integration (Optional)
- 📋 **Roadmap**: ConsolePort addon integration
  - Detect ConsolePort addon presence
  - Read controller bindings from ConsolePort when available
  - Hook into ConsolePort's action bar system
  - Sync recommendations with ConsolePort actions
  - Use ConsolePort's custom button mappings if configured
